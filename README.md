# desafio_dev_imoflow_elias_mateus

```bash
# Clone este repositório
$ git clone https://github.com/EliasMateu/desafio_dev_imoflow_elias_mateus.git

# Acesse o diretório
$ cd backend

# Instale as dependências
$ yarn

# Crie o banco de dados com o prisma
$ npx prisma generate

# Verefique se está rodando
$ npx prisma studio

# Execute o backend
$ yarn start:dev

# Acesse o diretório
$ cd frontend/imoflow

# Instale as dependências
$ yarn

# Execute o frontend
$ yarn dev

# O servidor será iniciado em <http://localhost:3000> para o backend e <http://localhost:5173> para o frontend
```
